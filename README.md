# Hackathon
웰니스 독서 서비스 럭키북키

## 프로젝트 소개
독서 후 기록을 통한 웰니스 달성<br>
독서를 통한 서비스 이용자들의 독서습관을 확립하고<br>
해당 책에 대한 내용을 자신의 현재 감정과 엮어 독서기록을 작성할 수 있게 함으로써<br>
서비스 이용자들의 멘탈 케어를 이끌 수 있는 서비스
